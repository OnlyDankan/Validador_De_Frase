Aplicação de console desenvolvida em C# que recebe uma frase digitada pelo usuário, realiza validação da entrada e do tamanho do texto, trata possíveis erros utilizando try/catch e salva a frase aceita em um arquivo de texto com registro de data e hora. O projeto também foi desenvolvido com o objetivo de praticar conceitos de organização, otimização e boas práticas no código, além de demonstrar o uso de manipulação de arquivos e controle de fluxo em aplicações de console.
